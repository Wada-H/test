AutoCutB
Subtract from the entire image the luminance value of the most common luminance value in each image (in most cases background noise) from the median or average, and multiply that peak by the Ext. value.

Subtract background from self slice
true: Perform AutoCutB for each image individually, false: Subtract the value calculated from the first image from all images.

AutoEnhance
Adjust the average brightness of subsequent images to match the average brightness of the first image.

Obtain (divide 100 <=) then ch2 / (ch1 + ch2) * bitMaxValue
Perform the calculation of ch2 / (ch1 + ch2) * bitMaxValue. If the brightness of both images is less than bitMaxValue/100, the ratio is set to 0. For example, for an 8-bit image, 255/100 = 2.55 -> If the value is less than 3, the calculated raito is set to 0.

ch2 / ch1 * 100 * exValue
Create an image by dividing ch2 by ch1, multiplying by 100, and then adjusting the brightness further by multiplying by an exValue.

Like IMD (should set Lut 'LIMD')
Create an 8-bit image in IMD format. To correctly represent this image, the LUT should be set to LIMD. IMD: A method to divide the ratio between Min-Max into 8 colors, further expressing the brightness in 32 gradations.

IMDwithHSB
A method to express the IMD format in HSB. The ratio is expressed in a continuous scale in HSB's Hue (purple-red, 0.25-1.0) between Min-Max. Brightness is expressed in a continuous scale for Saturation and Brightness in HSB, based on the proportion to the maximum brightness in that series (0.0-1.0).

Auto: Use the minimum and maximum ratio of that series.
Manual: Set an arbitrary range as the maximum and minimum ratio values. Ratios outside this range are expressed as the minimum or maximum.
C: Although the Hue is divided continuously, this value determines the division. For example, to divide into 8 colors, set it to 8.
